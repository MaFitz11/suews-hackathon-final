SUEWS Community Hackathon

UDA-city heat hazard and risk

This is the initial public showcase page for the hackathon repository. The full analysis will use SUEWS to model heat hazard across the ten synthetic UDA-city neighbourhoods and then bridge that environmental hazard to socio-economic heat risk.

Setup Status

  • Repository created from the official SUEWS hackathon template.
  • Task brief read and summarised for the workflow.
  • Challenge data identified as UDA-city, a synthetic hot-humid city with present and hotter-future forcing.
  • Live SUEWS example run is pending because the local suews-agent command is not currently available in this environment.

Next Analysis Step

Run the present hot-humid scenario for all ten neighbourhoods, calculate dangerous-heat hours from 2 m air temperature, and compare heat hazard with exposure and vulnerability.
